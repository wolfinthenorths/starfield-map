SOUTHERN WASTELAND MAP — v6

Что важно:
- На карте уже есть названия городов и номера бункеров.
- Поэтому поверх карты больше НЕ ставятся новые подписи, номера, маркеры или кружки.
- Города и бункеры нажимаются через невидимые кликабельные зоны.
- Бункер №05 удалён полностью.
- Районы появляются только после клика по выбранному городу.
- Районы занимают условную область выбранного города и разделены по типам цветом.

Как проверять:
1. Открой карту.
2. Нажми прямо по названию города, уже нарисованному на карте.
3. Должны появиться только районы этого города.
4. Нажми по уже нарисованному номеру бункера — откроется карточка бункера.
5. Если нужно убрать районы, нажми кнопку «Скрыть районы».
