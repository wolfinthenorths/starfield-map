СТАРТОВЫЙ НАБОР ДЛЯ ИНТЕРАКТИВНОЙ КАРТЫ SOUTHERN WASTELAND

Файлы:
- index.html — страница карты
- styles.css — оформление
- app.js — логика карты
- data.js — данные: города, бункеры, места, маршруты
- assets/wasteland-map.jpg — основная карта

Как загрузить на GitHub:
1. Создать репозиторий starfield-map.
2. Нажать Add file → Upload files.
3. Перетащить все файлы и папку assets из этой распакованной папки.
4. Commit changes.
5. Settings → Pages → Deploy from a branch → main → /(root) → Save.

Важно:
- Координаты сейчас черновые. Мы будем двигать точки после первого запуска.
- Значки сейчас технические, временные. Потом заменим на стилизованные под карту SVG-символы.
- Районы городов добавим следующим слоем после базового запуска.
