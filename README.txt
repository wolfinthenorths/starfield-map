SOUTHERN WASTELAND MAP — v4

Что изменено:
- используется новая чистая основа карты IMG_0611.PNG;
- названия городов НЕ дублируются поверх карты;
- номера бункеров НЕ дублируются поверх карты;
- бункеры открываются по клику на уже нарисованный номер;
- бункер №05 удалён;
- районы появляются только у выбранного города;
- районы занимают условную область города целиком, а не висят маленькими точками;
- подписи районов крупнее и на светлой плашке;
- цвета районов различаются по типу:
  жилые — охра,
  торговые — ржаво-коричневый,
  технические — серо-синий,
  культовые — бордово-фиолетовый,
  опасные — грязно-красный,
  административные — серо-зелёный,
  оборонительные — серо-коричневый,
  логистика — тёмная охра.

Как проверять:
1. Открой карту.
2. Нажми прямо на название/область города.
3. Карта приблизится, и появятся районы только этого города.
4. Нажми на район — справа откроется карточка.
5. Нажми на номер бункера на самой карте — справа откроется карточка бункера.

Если районная схема какого-то города не нравится по форме, лучше править именно форму city.shape и districts[].shape в data.js.
