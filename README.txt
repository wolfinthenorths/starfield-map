SOUTHERN WASTELAND MAP — v8

Что изменено:
- Карточки городов, районов и аномалий заполнены текстом из файла opisaniya_gorodov_raionov_anomaliy.
- Города и бункеры остались без дублей поверх карты.
- Бункер №05 отсутствует.
- Добавлен объект/аномалия «Сол-7».
- Аномалии кликабельны и открывают карточки.
- Границы районов больше не раздуваются наружу: вместо Catmull-Rom используется более безопасное сглаживание Chaikin, которое скругляет углы без сильного выползания за исходную форму.
- Районы прижаты ближе к городам и меньше заходят на воду/аномалии.

Проверка:
1. Нажмите на город — откроется его карточка из нового файла и появятся районы.
2. Нажмите на район — откроется карточка района из нового файла.
3. Нажмите на аномалию — откроется карточка аномалии.
4. Нажмите на номер бункера — откроется карточка бункера.
